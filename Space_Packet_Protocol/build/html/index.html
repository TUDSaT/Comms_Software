Space Packet¶

class SpacePacket.SpacePacket(packet_primary_header, packet_data_field)¶

A Space Packet is used to carry the data in the Space Packet Protocol.

__init__(packet_primary_header, packet_data_field)¶
Parameters:
  • packet_primary_header (PacketPrimaryHeader) – Consists of the Packet Version Number, Packet Identification Field, Packet Sequence Control Field and the Packet Data Length
  • packet_data_field (PacketDataField) – Mandatory and consists of the Packet Secondary Header and the User Data Field

Packet Data Field¶

class PacketDataField.PacketDataField(packet_secondary_header, user_data_field)¶

The Packet Data Field contains the optional Packet Secondary Header and the User Data Field.

__init__(packet_secondary_header, user_data_field)¶
Parameters:
  • packet_secondary_header (PacketSecondaryHeader) – The Secondary Header consists of either a Time Code Field only, an Ancillary Data Field only or a Time Code followed by an Ancillary Data Field
  • user_data_field (byte array) – Optional, if a Packet Secondary Header is present otherwise mandatory

__init__(packet_type, secondary_header_flag, application_process_identifier)¶

  • secondary_header_flag (bool) – Indicates if a Secondary Header is present or not

Packet Primary Header¶

class PacketPrimaryHeader.PacketPrimaryHeader(packet_version_number, packet_idenficiation, packet_sequence_control, packet_data_length)¶

The Packet Primary Header contains information about the Space Packet.

__init__(packet_version_number, packet_idenficiation, packet_sequence_control, packet_data_length)¶
Parameters:
  • packet_version_number (int) – Shall be set to 0
  • packet_idenficiation (PacketIdentification) – Consists of Packet Type, Secondary Header Flag and the API
  • packet_sequence_control (PacketSequenceControl) – Devided into Sequence Flag and the Packet Sequence Count or Packet Name
  • packet_data_length (int) – Byte count - 1

Packet Secondary Header¶

class PacketSecondaryHeader.PacketSecondaryHeader(time_code_field, ancillary_data_field)¶

The Packet Secondary Header consists of either a Time Code only or an Ancillary Data Field only or both.

__init__(time_code_field, ancillary_data_field)¶
Parameters:
  • time_code_field (str) – Shall consist of one of the CCSDS specified time codes
  • ancillary_data_field (str) – Not specified by the standard, may contain additional information

Packet Sequence Control¶

class PacketSequenceControl.PacketSequenceControl(sequence_flags, packet_sequence_count_or_packet_name)¶

The Packet Sequence Control contains Sequence Flags and information about the sequence count or name.

__init__(sequence_flags, packet_sequence_count_or_packet_name)¶
Parameters:
  • sequence_flags (int) – 0 for a continuation segment, 1 for first segment, 2 for last segment and 3 for unsegmented user data
  • packet_sequence_count_or_packet_name (integer or str) – For telemetry data sequence count, for telecommand either count or a name
